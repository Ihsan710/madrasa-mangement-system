# Madrasa Management System

A role-based management system for Madrasa administration.

## Access Links
- **Frontend (Application)**: [http://localhost:3000](http://localhost:3000)
- **Login Page**: [http://localhost:3000/login](http://localhost:3000/login)
- **Backend API**: [http://localhost:5000](http://localhost:5000)

## Credentials
- **Admin**: `admin` / `admin123`

## Features
- **Admin Panel**: Manage citizens, family members, and monthly fees.
- **Citizen Panel**: View family details and fee status.

## Setup
1. Backend: `cd mosque-backend && node server.js`
2. Frontend: `cd mosque-frontend && npm run dev`
