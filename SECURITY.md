# Security Policy

## Supported Versions

Please see the table below for supported versions of the Madrasa Management System.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to the maintainer team instead of opening a public issue. We will respond as quickly as possible to coordinate a fix and responsible disclosure.
